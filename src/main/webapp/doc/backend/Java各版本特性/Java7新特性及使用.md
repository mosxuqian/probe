# Java6新特性及使用

## 新特性列表

以下是Java7中的引入的部分新特性。关于Java7更详细的介绍可参考[这里](http://www.oracle.com/technetwork/java/javase/jdk7-relnotes-418459.html)。

- 集合语法增强
- switch支持String
- try-with-resources
- catch多个异常
- 实例创建类型推断
- 支持数字字面量下划线分割
- 二进制字面量
- 增强的文件操作
- fork/join 框架
- 其它
  - JDBC4.1规范
  - 支持动态类型语言
  - JSR341-Expression Language Specification
  - JSR203-More New I/O APIs for the Java Platform
  - 桌面客户端增强
